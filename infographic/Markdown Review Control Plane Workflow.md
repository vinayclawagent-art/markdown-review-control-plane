# Markdown Review Control Plane Workflow

```text
X signal
  ↓
Structured capture fields
  ↓
Agent-ready task packet
  ↓
Concrete artifact / patch / memo
  ↓
Evidence gate
  ↓
Promote, iterate, or hold
```

## Why it matters
Markdown is becoming an agent control surface, but teams still need Docs-like comments, suggestion mode, edit history, and CLI access without leaving git-readable files.
